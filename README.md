# blinkit-project
Power BI Sales Dashboard analyzing Blinkit grocery data with insights on total sales, outlet performance, item types, ratings, and fat content distribution.

## Project Overview

This project presents a Power BI dashboard analyzing sales performance of Blinkit – Myanmar’s Last Minute App.

The dashboard provides insights into:

💰 Total Sales: $1.20M

📦 Number of Items: 9K

⭐ Average Rating: 4

🏪 Outlet Performance by Size, Type, and Location

The goal of this project is to analyze retail sales performance and identify trends across outlet types, item categories, and fat content distribution.

## Dataset Information

Dataset includes the following fields:

Item Identifier

Item Type

Item Fat Content

Sales

Rating

Outlet Size

Outlet Location Type

Outlet Establishment Year

Item Visibility

Item Weight

## Dashboard Features

1️ Sales Overview

2️ Outlet Analysis

3️ Product Analysis

## Tools Used

Power BI Desktop

DAX (for calculated measures)

Data Cleaning & Transformation (Power Query)

## Key Insights

Tier 3 outlets generated the highest sales.

Supermarket Type1 has the highest total sales.

Low fat and regular fat items contribute differently across outlet tiers.

Sales trend increased significantly after 2018.
